## ✨ Features

<br/>

- 🎭 &nbsp; **Rich Animations** — Elastic, fade, slide & scale transitions throughout the entire app
- 🛒 &nbsp; **Global Cart State** — Add, remove, increment & decrement items synced across all screens in real time
- 🎬 &nbsp; **Animated Onboarding** — 3-page swipeable welcome with rotating orbit rings & floating emoji particles
- 🔐 &nbsp; **Auth Screens** — Sign In & Sign Up with animated focus glow, password toggle & social login buttons
- 💪 &nbsp; **Password Strength Meter** — Live animated bar checking length, uppercase, numbers & special characters
- 📱 &nbsp; **Bottom Navigation** — Animated tab bar with Home, Search, Orders and Profile tabs
- 🍽️ &nbsp; **5 Full Restaurants** — Each with complete menus, Popular tags, ratings & delivery times
- 💳 &nbsp; **Checkout & Payment** — Delivery address card, animated payment selector (Card / Apple Pay / Cash)
- 📍 &nbsp; **Order Tracking** — Live step tracker: Confirmed → Preparing → Out for Delivery → Delivered
- 🎨 &nbsp; **Custom UI** — Every widget hand-crafted. No external UI libraries needed.

## 🚀 Getting Started

<br/>

### Prerequisites

- Flutter SDK `>= 3.0.0`
- Dart SDK `>= 3.0.0`
- Android Studio or VS Code
- iOS Simulator or Android Emulator

<br/>

### Installation & Run

**1. Clone the repository**
```bash
git clone https://github.com/yourusername/foodie.git
cd foodie
```

**2. Get dependencies** *(nothing extra to install)*
```bash
flutter pub get
```

**3. Run the app**
```bash
flutter run
```

**4. Build for release**
```bash
# Android
flutter build apk --release

# iOS
flutter build ios --release
```
